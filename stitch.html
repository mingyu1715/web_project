CVE-2023-4567: Critical Vulnerability in Network Management System

A critical vulnerability has been identified in the Network Management System (NMS) software, potentially allowing remote attackers to execute arbitrary code. This flaw stems from improper input validation in the handling of network packets, leading to a buffer overflow. Successful exploitation could result in complete system compromise, including data theft and denial of service.

Affected Systems

This vulnerability affects all versions of the Network Management System (NMS) prior to version 2.5.3. Systems running on Windows, Linux, and macOS are susceptible.
